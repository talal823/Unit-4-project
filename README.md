# Dodge the Falling objects
In this game you have to dodge the falling red objects while trying to hit the green and yellow objects. Green one gives extra points while the yellow one gives a 5s immunity to the red object. 


Use the arrow keys to move your blue object. Three colored objects will fall from the sky : Red , Green and yellow. If you touch the red one game is over, Green ones give an instant 500 point bonus, and the Yellow ones will give a 5s immunity to the red objects. You can register and login, and try to beat the global highscore. 



The main program is in mainFrame.java.
